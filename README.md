# 家庭資金記帳工具

這是一個純前端、手機友善的家庭與個人記帳工具。

## 開啟網站

前往：<https://wong1022.github.io/expensetracker/>

## 資料安全

- 帳務資料只儲存在目前瀏覽器的 localStorage，不會提交到這個 repository。
- 清除瀏覽器資料、換手機或改用其他瀏覽器前，請先在「更多 → 備份與還原」匯出 JSON 備份。
- 第一次改用 Pages 網址時，請從舊版本匯出備份，再於新網址匯入。
- 本工具沒有帳號登入或雲端同步；請使用手機本身的鎖定功能保護資料。

## 加入手機主畫面

- iPhone：用 Safari 開啟網站，點分享，再選「加入主畫面」。
- Android：用 Chrome 開啟網站，從選單選擇「安裝應用程式」或「加到主畫面」。
